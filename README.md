<table align="center">

<h2>🚀 CreditRisk AI</h2>

<p>
An advanced <b>Machine Learning based Credit Risk Prediction System</b> built using 
<b>Python</b>, <b>Flask</b>, and <b>Scikit-Learn</b>.
</p>

<p>
The system predicts whether a customer is:
</p>

<ul>
<li>✅ Good Credit Risk</li>
<li>⚠️ Bad Credit Risk</li>
</ul>

<h3>🔥 Includes</h3>

<li>⚡ Feature Engineering</li>
<li>🤖 Logistic Regression Model</li>

<h1 align="center">🚀 Installation & Setup</h1>

<h3>1️⃣ Clone Repository</h3>

<pre>
git clone https://github.com/your-username/creditrisk-ai.git
</pre>

<h3>2️⃣ Install Dependencies</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>3️⃣ Run Application</h3>

<pre>
python app.py
</pre>

<h3>4️⃣ Open Browser</h3>

<pre>
http://127.0.0.1:5000
</pre>
